## Install packages

### Using `uv` (recommended)

```bash
# install uv
curl -LsSf https://astral.sh/uv/install.sh | sh
# other installation methods: https://docs.astral.sh/uv/#installation
uv sync
```

### Using `pip`

```bash
# OR
python -m venv .venv
source .venv/bin/activate
pip install .
```
